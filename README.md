## How to Run

1. Clone the repository:  
    git clone https://github.com/Burgula-Praharsha/OPTIFY.git
     
3. Navigate to the project directory:  
   cd OPTIFY
   
5. Install required dependencies:  
   pip install -r requirements.txt
  
6. Run the application:  
   python <filename> .py
